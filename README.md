# yahoo-charts

allows to compare visually different stocks on the same screen

## Project Setup

```sh
npm install
```

### Set env

create .env.scripts with YAHOO_API_KEY (see .env.scripts.example)

### load data

update settings in a ./config/

run 

```sh
npm run load-data
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```
